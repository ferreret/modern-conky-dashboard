## Widgets nuevos

### 🩺 HEALTH — `draw_sysstatus`
Monitorización de salud del sistema, cacheada a 120s.
- **systemd**: unidades caídas (user + system) → punto verde si 0, coral si hay.
- **journalctl -p 3 -S "1 hour ago"**: contador de errores recientes → w50 si 0, ámbar si <20, coral si >20.
- **reboot-required**: detecta `/var/run/reboot-required` → línea ámbar solo si procede.
- **docker**: contadores up/down/unhealthy con color según severidad (cyan / ámbar / coral).

Script nuevo: `scripts/sysstatus.sh`. Ejecuta sin sudo — `systemctl --failed`, `journalctl -p 3` (accesible como usuario) y `docker info`.

### 🌳 GIT — `draw_gitstatus`
Estado multi-repo cacheado a 60s.
- Para cada repo configurado: branch actual, nº de archivos modificados, commits ahead/behind del upstream.
- Soporte de alias con formato `ruta|alias` en el archivo de config.
- Color del punto: verde (clean), ámbar (dirty), cyan (solo desincronizado).
- Máximo 8 repos mostrados.

Script: `scripts/gitstatus.sh`. Config: `modern/.gitrepos` (gitignorado por privacidad; plantilla en `.gitrepos.example`).

Repos actualmente vigilados por el usuario:
- `~/.config/conky/modern`
- `FlexiPy|DocScan`
- `portolio|Portfolio`
- `inversiones-tracker|Inversiones`

### ☀️ Weather v2 — `draw_weather`
Rediseño completo del widget meteorológico, pasando de 3 líneas bajo el reloj a un widget dedicado a la derecha de Mazinger.

Ampliación del script `weather.sh`:
- Nuevos campos: `CODE` (weather code numérico), `WIND_DEG`, `SUNRISE`, `SUNSET`, `MOON`.
- **Forecast hourly**: 8 puntos cada 3h cubriendo las próximas 24h (`H0..H7_T/P/C/H`).
- Forecast diario ampliado con código para colorear.

Render:
- Temperatura grande (68s), "feels like", DESC, LOCATION.
- **Sparkline de temperatura 24h** con puntos en cada muestra + etiquetas min/max.
- **Barras de precipitación** debajo de la sparkline (solo si `pmax > 0`).
- **Badge UV** con 5 niveles de color (verde → ámbar → coral → pink → purple) + texto ("low", "moderate"…).
- Sunrise ↑ + Sunset ↓.
- Wind con dirección, humidity, pressure.
- Forecast 3 días con punto de color por weather code.

Mini bajo el reloj reducido a una línea (`23° Sunny`) para no duplicar.

## Polish visual

### Sombra de texto para legibilidad
La función `txt()` pinta ahora cada string dos veces:
1. Sombra negra alpha 0.65, offset proporcional (`sz * 0.06`).
2. Color real encima.

Aplica automáticamente a los 17 widgets. Probado y aprobado por el usuario.

### Alpha boost de grises
El sombreado solo mejoró "un poco". Combinado con subida de alpha en la paleta:
- `w70: 0.70 → 0.85`
- `w50: 0.50 → 0.68`
- `w35: 0.35 → 0.52`

Resultado: "mejor" confirmado por el usuario.
